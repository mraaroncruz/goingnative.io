Android and iOS sittin' in a tree

Screencast courses to teach you how to create native mobile apps using the Android SDK and iOS's Cocoa Touch framework.

Also learn API development and the architecture and design you will need to make your apps easily portable back and forth

Android

We will start from scratch, building Android apps step by step, thinking of portability first. I like to think of this style as Portability First Development. We will come out of this with a working, useful and easily modifiable Android app.

iOS

We will build the same app using Objective C and possibly the new Swift programming language. You will be able to learn best practices to make this process much less painful than you can believe.

The API

We need a way for the apps to manage data so we will spend some time creating an API for both apps to communicate with. If you want to up your hourly rate as a developer, these skills will push you to the next level.
